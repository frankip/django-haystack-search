### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:frankip/django-haystack-search.git
   ```
2. Add a virtual environment and activate
   ```sh
   python3 -m venv venve

   activating

   source venv/bin/activate
   ```
3. Install all dependencies
   ```py
   pip install -r requirements.txt
   ```
4. run migrate 
   ```py
   python manage.py migrate
   ```
5. load data to the database
   ```py
   python3 init_db.py

   ```
6. Run servers
   ```py
   python manage.py runserver
   ```
